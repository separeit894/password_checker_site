## Установка

```bash
git clone --recursive https://github.com/separeit894/password_checker_site
```

## Запуск

### Первый способ

В браузере через 

```bash
file:///путь/до/файла/index.html
```

### Второй способ

1. ПКМ по файлу __index.html__
2. Открыть в браузере (на ваше усмотрение)
